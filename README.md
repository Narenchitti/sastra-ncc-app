# SASTRA NCC Boys Contingent App

This is the Full Stack Web Application for the SASTRA NCC Boys Contingent.
Built with **Next.js 14**, **Tailwind CSS**, and **TypeScript**.

## Getting Started

First, install dependencies (only needed once):

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
